About

What is DartMark? I'm glad you asked. DartMark is a WYSIWYG with a focus on speed and prototyping. The main benefits of it are:

  1. Speed (all actions can be performed with just a keyboard)
  2. Ease-of-use (no need to switch back and forth between a CSS editor and previewing anymore)
  3. Configurability (DartMark will let you define your own keyboard shortcuts and special JavaScript function macros)
